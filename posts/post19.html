Battery Power

In order to make a more realistic prototype, the physical component needed to be battery powered. If it was powered from a usb, then it would not be portable. In order to replace the power source I connected the hardware to a battery using an on off switch.

In order to be able to easily access this switch and change the battery the physical component would need some sort of opening made from a zipper, snaps, or velcro during actual production.
